## 역할

가족 구성원이 사진과 함께 기록을 남기고, 부모/자식 모두 음성(STT) 또는 텍스트로 댓글을 추가할 수 있는 공유 캘린더입니다.

## 핵심 기능

1. **캘린더 조회**: 월/주 단위 캘린더 뷰
2. **사진 업로드**: 특정 날짜에 사진 추가 (가족 행사, 일상, 추억)
3. **댓글 작성**:
   - **부모**: 음성 녹음 → STT로 텍스트 변환 → 댓글 저장, 또는 직접 텍스트 입력
   - **자식**: 음성 녹음 → STT로 텍스트 변환 → 댓글 저장, 또는 직접 텍스트 입력
4. **댓글 표시**: 각 날짜/사진에 작성자별 댓글 목록 표시
5. **음성 원본 보존**: STT 변환 원본 음성 파일도 함께 저장 (추후 재생 가능)

## 데이터 구조 (제안)

### CalendarEntry (캘린더 항목)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 항목 ID |
| family_id | UUID | 가족 ID |
| date | Date | 캘린더 날짜 |
| title | String | 제목 (예: "손자 첫돌") |
| image_url | String | 사진 파일 경로 |
| created_by | UUID | 작성자 ID |
| created_at | DateTime | 생성 일시 |

### CalendarComment (캘린더 댓글)

| 필드 | 타입 | 설명 |
|------|------|------|
| id | UUID | 댓글 ID |
| entry_id | UUID | 캘린더 항목 ID |
| member_id | UUID | 작성자 ID |
| content | Text | 댓글 내용 (텍스트) |
| input_method | Enum | stt / text |
| audio_file_path | String | 원본 음성 파일 경로 (nullable) |
| created_at | DateTime | 생성 일시 |

## 다른 모듈과의 관계

- **server**: 캘린더 항목/댓글 CRUD API, 사진 업로드 API
- **stt**: 음성 댓글 작성 시 STT 모듈 호출
- **app**: 부모 모드/자식 모드 모두에서 캘린더 탭 접근
- **shared**: 캘린더 데이터 스키마 공유

## 확장 아이디어

- 사진 자동 태깅: AI로 사진 인물/장면 자동 분류
- 이벤트 알림: 특정 날짜(생일, 기념일) 자동 표시
- 캘린더에서 스토리북으로: 특정 기간의 캘린더 항목을 모아 스토리북 페이지로 자동 변환
- 사진 슬라이드쇼: 월별 사진 슬라이드쇼 자동 생성
- 댓글 감정 분석: 댓글에서 감정을 추출하여 캘린더에 이모지 표시
