# Question Engine (질문 생성 엔진)

## 역할

자식이 부모님에게 할 질문을 직접 작성하거나, 시스템이 자동으로 질문을 생성합니다.

## 구성해야 할 파일

| 파일 | 설명 |
|------|------|
| `manual_question.py` | 자식이 직접 작성하는 질문 입력 로직 |
| `auto_question.py` | 카테고리/템플릿 기반 자동 질문 생성 |
| `question_templates.py` | 질문 템플릿 데이터 (카테고리별) |
| `ai_question.py` | LLM 기반 맞춤 질문 생성 (Phase 2) |
| `requirements.txt` | Python 의존성 |

## 질문 카테고리 (제안)

| 카테고리 | 예시 질문 |
|----------|-----------|
| 유년기 | "어린 시절 가장 행복했던 기억은 무엇인가요?" |
| 청춘 | "스무 살 적, 가장 큰 꿈은 무엇이었나요?" |
| 사랑 | "어머니/아버지를 처음 만났을 때 어떤 느낌이셨나요?" |
| 인생 | "살면서 가장 힘들었던 시기는 언제였나요?" |
| 가치관 | "인생에서 가장 중요하게 생각하는 가치는 무엇인가요?" |
| 가족 | "우리 가족에게 해주고 싶은 말씀이 있나요?" |
| 음식 | "엄마의 특별한 요리 레시피를 알려주실 수 있나요?" |
| 계절성 | "올해 가장 감사했던 일은 무엇인가요?" |

## 다른 모듈과의 관계

- **child-app**: 자식 앱에서 질문 작성 UI 제공, 이 모듈의 API 호출
- **server**: 생성된 질문을 서버 API(`POST /api/questions`)로 전송
- **parent-app**: 서버를 통해 부모 앱으로 질문 전달
- **shared**: 질문 데이터 스키마 공유

## 개발 시 구현해야 할 내용

- [ ] 수동 질문 작성 함수 (`create_question(content, from_id, to_id) -> Question`)
- [ ] 카테고리별 질문 템플릿 데이터 구축
- [ ] 자동 질문 생성 함수 (`generate_question(category, context) -> Question`)
- [ ] 질문 중복 방지 로직 (이미 답변한 질문 재생성 방지)
- [ ] 서버 API 연동 (질문 전송)
- [ ] (Phase 2) LLM 기반 맞춤 질문 생성 - 이전 응답을 컨텍스트로 활용

## 확장 아이디어

- 이전 응답을 분석한 후속 질문 자동 제안 (대화형 질문 확장)
- 계절/명절 맞춤 질문 패키지 (어버이날, 추석, 설날)
- 질문 난이도 조절 (가벼운 질문 → 깊은 질문으로 단계적 진행)
- 자식이 선택할 수 있는 질문 후보 추천 시스템
