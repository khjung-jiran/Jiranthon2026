## 역할

자식이 보낸 질문 텍스트를 부모님 앱에서 음성으로 발화합니다.
부모님이 글을 읽기 어려워하시는 경우, 질문을 음성으로 들려줍니다.

## 구성해야 할 파일

| 파일 | 설명 |

## 다른 모듈과의 관계

- **parent-app**: 부모 앱이 서버에서 질문을 수신하면 TTS 모듈을 호출하여 발화
- **server**: 서버에서 전달된 질문 텍스트를 TTS 입력으로 사용
- **storybook**: 스토리북을 오디오북 형태로 생성할 때 활용

## 개발 시 구현해야 할 내용

- [ ] 텍스트 입력 → 음성 파일(mp3/wav) 생성 함수 (`text_to_speech(text: str) -> bytes`)
- [ ] 한국어 음성 지원 (문장 끊김, 자연스러운 억양)
- [ ] 부모 앱에서 재생 가능한 형태로 반환 (파일 경로 또는 스트리밍)
- [ ] 음성 속도/볼륨 조절 옵션 (고령자를 위한 느린 속도 기본값)
- [ ] 질문 길이에 따른 분할 발화 (긴 질문을 문장 단위로 나누어 발화)

## 확장 아이디어

- Voice Cloning: 부모님 목소리로 스토리북 오디오북 생성
- 감정 적용: 질문에 감정을 담아 자연스러운 발화
- 배경음악: 스토리북 오디오북에 배경음악 자동 믹싱
