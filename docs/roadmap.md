## Phase 2: 고도화

### AI 질문 생성

- **LLM 기반 맞춤 질문**: GPT/Claude API를 활용하여 부모님 연령, 가족 관계, 계절, 이벤트 등을 반영한 맞춤형 질문 자동 생성
- **대화형 질문 확장**: 이전 응답을 분석하여 후속 질문 자동 제안 ("아버님이 말씀하신 그 시절에 가장 친했던 친구는 누구였나요?")
- **질문 추천 시스템**: 자식이 선택할 수 있는 질문 후보를 카테고리별로 추천

### 음성 고도화

- **화자 분리 (Speaker Diarization)**: 부모님과 다른 사람의 음성을 구분하여 인식 정확도 향상
- **감정 분석**: 음성에서 감정을 분석하여 스토리북에 감정 태그 추가
- **음성 친숙화**: 부모님 목소리로 TTS 발화 (Voice Cloning)하여 자식이 부모님 목소리로 스토리북을 들을 수 있게
- **방언 인식**: 한국어 방언(경상도, 전라도, 제주도 등) 인식 지원

### 스토리북 고도화

- **AI 이미지 생성**: 응답 내용을 바탕으로 DALL-E/Stable Diffusion으로 삽화 자동 생성
- **다양한 출력 포맷**: PDF, EPUB, 웹 갤러리, 인쇄용 화질
- **테마/디자인 템플릿**: 감성, 모던, 클래식 등 다양한 스토리북 디자인
- **음성 스토리북**: TTS로 전체 스토리북을 오디오북 형태로 생성

## Phase 3: 플랫폼화

### 소셜 기능

- **가족 공유**: 스토리북을 가족 그룹 내에서 공유 및 협업 편집
- **세대 간 스토리북**: 조부모 → 부모 → 자식으로 이어지는 3세대 스토리북
- **추억 타임라인**: Q&A를 시간순으로 나열한 가족 타임라인 뷰

### 알림 및 참여 유도

- **정기 질문 알림**: 매주/매월 자동으로 질문 전송 및 푸시 알림
- **게이미피케이션**: 연속 응답 보너스, 스토리북 완성 배지 등 참여 유도
- **시즌 이벤트**: 어버이날, 추석, 설날 등 명절 맞춤 질문 패키지

### 접근성

- **고령자 친화적 UI**: 큰 글씨, 고대비, 음성 안내, 단순 네비게이션
- **음성 명령**: 부모 앱을 음성으로 조작 ("다음 질문 들려줘", "다시 말할게")
- **다국어 지원**: 한국어 + 영어 + 일본어 + 중국어

---

## Phase 4: 데이터 가치화

### 인사이트 분석

- **가족 가치관 분석**: 응답 데이터에서 가족의 핵심 가치관, 인생 철학 추출
- **세대 비교 분석**: 부모와 자식의 응답을 비교하여 세대 간 차이/공통점 시각화
- **워드 클라우드**: 부모님이 자주 사용하는 단어 분석

### 유전/계승 기능

- **가족 레시피북**: 음식 관련 Q&A를 모아 가족 레시피북 자동 생성
- **인생 지혜 카드**: 핵심 응답을 카드 형태로 추출하여 공유 가능
- **유언/유품 기록**: 법적 효력은 없지만, 부모님의 유언 및 유품 관련 기록 보존

## 기술적 확장 아이디어

| 아이디어 | 기술 | 우선순위 |
|----------|------|----------|
| 실시간 질문-응답 (화상 통화) | WebRTC | 중 |
| 음성 감정 분석 | TensorFlow/PyTorch | 중 |
| AI 이미지 생성 삽화 | DALL-E / Stable Diffusion | 높음 |
| Voice Cloning TTS | Coqui TTS / Azure Custom Voice | 중 |
| 모바일 푸시 알림 | FCM / APNs | 높음 |
| 오프라인 동작 | PWA + 로컬 DB | 낮음 |
| 블록체인 기반 기록 보존 | IPFS + 스마트 컨트랙트 | 낮음 |
